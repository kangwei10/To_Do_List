# To_Do_List
This is a To-Do-List webapp coded using React.js and .Net

## To run the app
1. cd ToDoListApp
2. run "dotnet run"
3. Navigate to ToDoListApp.Client/.env
4. Change VITE_API_UR to your local url
5. cd ToDoListApp.Client
6. run "npm start dev"
7. Double click your local url

## To test
1. cd ToDolistApp.Tests
2. run "dotnet test"
