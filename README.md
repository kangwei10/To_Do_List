# To_Do_List
This is a To-Do-List webapp coded using React.js and .Net

## To run the app
1. cd ToDoListApp
2. run "dotnet run"
3. Navigate to ToDoListApp.Client/.env
4. Change VITE_API_UR to your local url
5. Stop the terminal
6. cd ToDoListApp.Client
7. run "npm start dev"
8. Copy your local url and paste on ToDoListApp/Program.cs line 14
9. Stop the terminal and redo step 1, 2, 6, 7
10. Double click your local url from step 7 

## To test
1. cd ToDolistApp.Tests
2. run "dotnet test"
